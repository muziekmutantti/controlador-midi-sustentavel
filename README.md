🎛️ Controladores MIDI Sustentáveis

Projeto de desenvolvimento de controladores MIDI customizados, construídos com Arduino, componentes reciclados e tecnologias open source.

👤 Autor
Vagné L.
Artista sonoro e pesquisador baiano que investiga frequências meditativas, drones e paisagens sonoras afro-amazônicas e periféricas. Seu trabalho articula tecnologia e ancestralidade por meio da metarreciclagem, criando instrumentos eletrônicos não convencionais aliados à manipulação sonora digital e gravação de campo.

Fundador do selo Muziek Mutantti, atua no desenvolvimento de arte sonora contemporânea afroindígena, utilizando tecnologias abertas como Arduino e softwares musicais livres em projetos como:

Periferias AfroExperimentais

📌 Sobre o Projeto

O Controladores MIDI Sustentáveis é uma iniciativa que propõe a criação de dispositivos musicais acessíveis, sustentáveis e replicáveis.

Os controladores são desenvolvidos com foco em:

♻️ Sustentabilidade (uso de componentes reciclados)
🔓 Cultura open source
🎶 Democratização da produção musical
🧠 Educação tecnológica em contextos periféricos
⚙️ Especificações Técnicas

O dispositivo possui uma arquitetura simples e eficiente:

4 botões digitais (inputs)
2 potenciômetros analógicos
1 LED indicador (feedback visual)
Comunicação via protocolo MIDI padrão
Baseado em Arduino
🎚️ Funcionalidades
Controle de parâmetros em DAWs (Ableton Live, FL Studio, Reaper, etc.)
Integração com softwares musicais open source
Compatibilidade com dispositivos mobile
Resposta rápida e precisa para performance ao vivo e produção musical
🧩 Tecnologias Utilizadas
Arduino (hardware e firmware)
Protocolo MIDI
Softwares musicais open source
Técnicas de metarreciclagem eletrônica
🌱 Sustentabilidade

O projeto adota práticas de metarreciclagem, reutilizando componentes eletrônicos descartados para reduzir impacto ambiental e ampliar o acesso à tecnologia.

Essa abordagem:

Diminui o lixo eletrônico
Reduz custos de produção
Incentiva autonomia tecnológica
🎓 Formação e Aprendizagem

O projeto inclui:

Montagem do controlador
Programação em Arduino
Configuração MIDI
Integração com softwares musicais
🚀 Aplicações
Produção musical independente
Performances ao vivo
Educação tecnológica e musical
Oficinas em escolas públicas e projetos culturais

📦 Como Replicar
Obtenha um Arduino (Leonardo)
Reúna componentes:
Botões
Potenciômetros B10K
LED
Resistores 220 ohms
Faça a montagem do circuito
Carregue o código no Arduino
Conecte ao software musical
